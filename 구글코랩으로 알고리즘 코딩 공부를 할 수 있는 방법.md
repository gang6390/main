구글 코랩(Google Colab)은 원래 데이터 분석이나 AI 연구용으로 많이 쓰이지만, **알고리즘 공부와 문제 풀이 기록용**으로도 매우 훌륭한 도구입니다.

특히 코랩은 **'코드 + 설명(마크다운) + 실행 결과'**를 한 장의 문서처럼 저장할 수 있어, 나중에 복습할 때 최고입니다. 코랩을 활용해 효율적으로 알고리즘을 공부하는 5가지 팁을 추천해 드립니다.

---

### 1. 백준용 입력(Input) 처리 "꿀팁"
백준 문제는 보통 대량의 데이터를 입력받아야 합니다. 코랩에서 매번 `input()`을 입력하는 건 불편하므로, 아래 방법을 추천합니다.

*   **방법: 테스트 케이스를 텍스트 파일로 저장하고 읽기**
    코드 셀 가장 위에 아래 코드를 넣으면, 백준의 예제 입력을 파일로 만들고 이를 소스 코드가 읽게 할 수 있습니다.

    ```python
    # 1. 예제 입력을 input.txt라는 파일로 만듭니다.
    %%writefile input.txt
    5
    10 20 30 40 50

    # 2. sys.stdin을 이 파일로 교체합니다. (백준 제출 시에는 이 부분만 지우면 됨)
    import sys
    sys.stdin = open('input.txt', 'r')

    # 3. 이제 평소처럼 코딩합니다.
    n = int(sys.stdin.readline())
    data = list(map(int, sys.stdin.readline().split()))
    print(sum(data))
    ```
    *   **장점:** 백준에 제출할 때는 맨 위 파일 읽기 부분만 빼고 복사해서 제출하면 되므로 매우 편리합니다.

### 2. '마크다운'을 활용한 오답 노트 작성
코랩의 가장 큰 장점입니다. 단순히 코드만 적지 말고, **코드 셀 위에 마크다운 셀**을 추가하세요.

*   **추천 기록 항목:**
    *   **문제 링크:** 백준 번호와 제목
    *   **핵심 아이디어:** "이 문제는 BFS를 써야 함", "메모이제이션으로 중복 계산 방지" 등
    *   **시간 복잡도:** $O(N \log N)$ 등 분석 내용
    *   **실수했던 점:** "인덱스 범위를 0부터가 아니라 1부터 잡아서 틀렸음"

### 4. 시각화 도구 활용 (그래프/트리)
알고리즘은 눈에 보이지 않아 어렵습니다. 코랩에서는 파이썬 라이브러리를 사용해 그래프나 트리 구조를 시각화해볼 수 있습니다.

*   **NetworkX 라이브러리:** 그래프 알고리즘(최단 경로 등)을 공부할 때 내가 만든 인접 리스트가 맞는지 그림으로 그려볼 수 있습니다.
*   **Matplotlib:** 정렬 알고리즘의 진행 과정을 그래프로 그려보는 등 시각적 이해가 가능합니다.
