## 온라인 환경에서 알고리즘 코딩 공부 로드맵

온라인 환경에서 알고리즘 코딩 공부를 효율적으로 하기 위한 **단계별 로드맵과 최고의 리소스**를 정리해 드립니다. 이 방법은 수많은 합격자와 현업 개발자들이 검증한 방식입니다.

---

### 1단계: 기본 언어 마스터 및 기초 자료구조 이해
알고리즘은 결국 '도구'를 사용하는 법입니다. 하나의 언어를 깊게 파는 것이 중요합니다.

*   **추천 언어:** **Python**(배우기 쉽고 라이브러리가 강력함), **C++**(속도가 빠르고 코딩 테스트의 표준), **Java**(기업 코테에서 많이 쓰임).
*   **학습 내용:** 배열(Array), 리스트(List), 스택(Stack), 큐(Queue), 해시 테이블(Hash Table)의 기본 원리와 구현법.
*   **추천 리소스:**
    *   **유튜브 '나동빈(ndb796)':** [이것이 취업을 위한 코딩 테스트다](https://www.youtube.com/playlist?list=PLRx0vPvlEmdAghTr5mXQxG29qaQNmD6n7) (입문자 필수 코스)
    *   **인프런/유데미:** 기초 자료구조 강의.

### 2단계: 필수 알고리즘 유형 정복
코딩 테스트에 나오는 유형은 어느 정도 정해져 있습니다. 다음 순서로 공부하세요.

1.  **그리디(Greedy):** 현재 상황에서 최선의 선택.
2.  **구현(Implementation):** 문제 조건을 코드로 옮기기 (시뮬레이션, 완전 탐색).
3.  **DFS/BFS:** 그래프 탐색의 기초 (가장 중요).
4.  **정렬/이진 탐색:** 데이터 찾기 최적화.
5.  **다이나믹 프로그래밍(DP):** 작은 문제로 큰 문제 해결 (가장 어려움).
6.  **최단 경로(Dijkstra):** 노드 간 최단 거리.

### 3단계: 문제 풀이 플랫폼 활용 (가장 중요)
이론만 공부하면 실력이 늘지 않습니다. 직접 풀어봐야 합니다.

*   **[백준(Baekjoon Online Judge)](https://www.acmicpc.net/):** 국내 최대 규모. **'solved.ac'** 크롬 확장 프로그램을 설치하여 난이도(브론즈~플래티넘)를 확인하며 푸세요.
*   **[프로그래머스(Programmers)](https://programmers.co.kr/):** 카카오, 라인 등 국내 대기업 기출문제가 많고 웹 IDE 환경이 실제 시험과 비슷합니다.
*   **[LeetCode](https://leetcode.com/):** 글로벌 표준. 외국계 기업이나 구글/메타 등을 준비한다면 필수입니다.
*   **[Codeforces](https://codeforces.com/):** 알고리즘 대회(CP) 지망생을 위한 실시간 경쟁 사이트.

### 4단계: '진짜' 실력을 높이는 공부 전략 (Tip)

1.  **한 문제에 너무 오래 매달리지 마세요:**
    *   초보자라면 **30분~1시간** 고민해도 안 풀리면 과감히 답안(풀이)을 보세요.
    *   답을 본 후에는 반드시 **직접 다시 코드를 짜보고**, 왜 그 로직이 나왔는지 이해해야 합니다.
2.  **복습의 힘 (오답 노트):**
    *   풀었던 문제라도 일주일 뒤에 다시 풀면 못 푸는 경우가 많습니다. 깃허브(GitHub)나 블로그(Tistory, Velog)에 풀이 과정을 기록하세요.
3.  **양보다는 질:**
    *   하루에 10문제를 대충 푸는 것보다, 1문제를 풀어도 **시간 복잡도(Time Complexity)**를 최적화하는 고민을 하는 것이 훨씬 도움됩니다.
4.  **Solved.ac '클래스' 따라가기:**
    *   무엇부터 풀지 모르겠다면 Solved.ac의 **'CLASS' 메뉴**를 순서대로 정복하세요. 로드맵이 잘 짜여 있습니다.

### 5단계: 실전 감각 익히기
*   **정기적인 대회 참여:** 프로그래머스 월간 코드 챌린지나 백준의 각종 대회를 통해 긴장된 상태에서 푸는 연습을 하세요.
*   **타이머 설정:** 실제 시험처럼 1문제당 40~60분 정도 시간을 재고 푸는 연습을 하세요.
